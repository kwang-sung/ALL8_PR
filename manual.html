대시보드 — 전체 현황 한눈에

처음 들어오면 보이는 화면이에요. 지금 내 계정이 어떤 상태인지 한눈에 볼 수 있어요.

항목설명
수집된 상품지금까지 담아온 상품 총 개수
판매 중인 상품마켓에 현재 올라가 있는 상품 수
크레딧 잔액서버 수집·AI 기능 사용 시 차감되는 포인트. 부족하면 충전하세요.
재고 연동 중소싱처 품절 여부를 자동으로 감시 중인 상품 수
업로드 중지금 마켓에 올리고 있는 상품 수 (잠깐 기다리면 완료돼요)
에러뭔가 잘못된 상품 수. 클릭하면 원인을 볼 수 있어요.
📌 크레딧이 뭐예요?
서버 수집이나 AI 기능을 쓸 때 소모되는 포인트예요. 기본 브라우저 수집은 크레딧이 들지 않아요. 크레딧이 다 떨어지면 서버 수집이 안 되니 대시보드에서 잔액을 가끔 확인하세요.

상품 수집 — 소싱처에서 상품 가져오기

소싱 사이트에서 팔고 싶은 상품을 ALL8으로 가져오는 단계예요. 방법이 몇 가지 있는데 하나씩 설명할게요.

① 상품 1개 담기 (기본)
  1. 담고 싶은 상품 페이지를 크롬으로 열기 알리익스프레스, 타오바오, 1688, 아마존 등 어디서든 가능해요.
  2. 오른쪽 ALL8 사이드패널 확인 사이드패널이 안 보이면 브라우저 오른쪽 위 ALL8 아이콘을 클릭하세요.
  3. 사이드패널에서 이 상품 수집 클릭 10~30초 정도 기다리면 상품 정보를 자동으로 가져와요.
② 여러 개 한꺼번에 담기
  • 카테고리 페이지나 검색 결과 페이지로 이동하세요.
  • 각 상품 위에 체크박스가 뜹니다. 원하는 것들을 체크하세요.
  • 사이드패널에서 수집 시작을 누르면 선택한 상품이 한꺼번에 담겨요.
③ URL·엑셀로 수동 추가

확장 프로그램 없이도 링크를 직접 입력해서 상품을 담을 수 있어요.

  • URL 직접 입력: 사이드패널 또는 상품 목록 화면에서 + URL로 추가 클릭 → 링크 붙여넣기
  • 엑셀 파일로 대량 추가: + 엑셀로 추가 클릭 → URL이 담긴 엑셀/CSV 파일 업로드 → 한 번에 수백 개도 가능해요.
📋 엑셀 형식
A열에 URL만 적어두면 돼요. 헤더(제목 행)는 없어도 되고 있어도 돼요.
④ 브라우저 수집 vs 서버 수집

💻 브라우저 수집 (무료)

  • 내 크롬이 직접 상품 페이지를 열어서 가져와요
  • 크레딧 소모 없음
  • 내 컴퓨터가 켜져 있어야 해요
  • 처음 시작하는 분께 추천

☁️ 서버 수집 (크레딧 소모)

  • ALL8 서버가 대신 가져다줘요
  • 컴퓨터 꺼져도 수집돼요
  • 더 빠르고 안정적이에요
  • 크레딧이 조금 차감돼요

처음엔 브라우저 수집으로 충분해요. 상품이 많아지거나 자동화가 필요할 때 서버 수집을 써보세요.

⑤ 재고 연동 — 품절 자동 감지

소싱처에서 상품이 품절되면 내 마켓에도 빨리 반영해야 해요. 안 그러면 고객이 주문했는데 못 보내는 상황이 생겨요.

  • 상품 목록에서 여러 개 선택 → 재고 연동 설정 클릭
  • ALL8이 주기적으로 소싱처를 확인해서 품절이면 자동으로 판매 중지해줘요.
  • 상품이 10개 이상 쌓이면 꼭 켜두는 게 좋아요.
⚠️ 주의
재고 연동은 소싱처 사이트가 바뀌거나 로그인이 필요한 경우 제대로 동작하지 않을 수 있어요. 가끔 직접 확인도 병행하세요.

알리익스프레스 타오바오 티몰 1688 알리바바 아마존 미국/일본 쿠팡 스마트스토어 테무 그 외 다수 지원

전체 지원 목록은 확장 프로그램 설정(⚙) → 지원 사이트에서 확인하세요.

상품 편집 — 마켓에 올리기 전 설정

상품을 클릭하면 편집 화면이 열려요. 위에서 아래로 채워나가면 됩니다. 어렵지 않아요.

1) 마켓 선택

어느 마켓에 올릴지 선택해요. 미리 계정을 연결해뒀어야 보여요.

쿠팡 스마트스토어 11번가 G마켓 옥션 인터파크 쇼피 그 외
💡 여러 마켓 동시에 선택 가능
마켓을 여러 개 선택하면 한 번에 여러 곳에 올릴 수 있어요.
2) 상품명
  • AI가 소싱처 상품명을 자동으로 번역·정리해줘요.
  • 마음에 안 들면 직접 수정하세요. 클릭하면 편집할 수 있어요.
  • 금지어가 포함되어 있으면 자동으로 걸러줘요 (금지어 설정에서 미리 등록 필요).
  • 마켓마다 상품명 글자 수 제한이 있어요. 쿠팡은 100자, 스마트스토어는 100자 이내.
3) 카테고리
  • AI가 상품에 맞는 카테고리를 자동으로 추천해줘요.
  • 틀렸다 싶으면 검색창에서 직접 찾아서 바꿀 수 있어요.
  • 마켓마다 카테고리 체계가 달라서 각각 따로 선택해야 해요.
⚠️ 카테고리가 잘못되면
마켓에서 등록을 거부하거나 잘못된 곳에 올라갈 수 있어요. AI 추천을 한 번 확인하고 넘어가세요.
4) 가격 설정
항목설명
원가 (소싱 단가)소싱처에서 실제로 구매하는 가격. 자동으로 가져오는데, 환율 등이 반영 안 됐으면 수동 수정하세요.
마진율 (%)원가에서 얼마나 남길 건지 비율. 예: 원가 10,000원 + 마진 30% = 판매가 13,000원.
추가 비용배송비, 수수료 등 원가 외 드는 비용.
판매가위 항목을 기반으로 자동 계산돼요. 직접 입력해서 바꿀 수도 있어요.
📌 프리셋에 저장하면 편해요
마진율을 매번 입력하기 귀찮으면 프리셋으로 저장해두세요. 다음엔 클릭 한 번으로 끝나요.
5) 배송 설정
항목설명
배송비고객에게 받을 배송비. 무료배송이면 0원 입력.
배송 방법택배, 직접 배송 등 선택.
출고지상품이 출발하는 주소. 마켓에 등록된 주소와 같으면 그대로 두세요.
반품지반품 받을 주소. 보통 출고지와 같아요.
반품 배송비고객이 반품할 때 내는 배송비.
6) 이미지
  • 소싱처 상품 이미지를 자동으로 가져와요.
  • 순서 변경: 이미지를 드래그해서 순서를 바꿀 수 있어요. 첫 번째 이미지가 대표 이미지가 돼요.
  • 삭제: 불필요한 이미지 위에 마우스를 올리면 삭제 버튼이 뜨거요.
  • 추가: 내 컴퓨터에서 이미지를 직접 올릴 수도 있어요.
  • 마켓마다 이미지 개수 제한이 있어요 (쿠팡 10장, 스마트스토어 10장 등).
💡 대표 이미지가 중요해요
검색 결과에서 처음 보이는 이미지예요. 깔끔하고 상품이 잘 보이는 이미지를 첫 번째로 두세요.
7) 상세 설명 (상세페이지)
  • AI가 소싱처 상세페이지를 번역·정리해서 자동으로 만들어줘요.
  • 마음에 안 들면 직접 수정할 수 있어요.
  • HTML 편집도 가능해요 (HTML 탭으로 전환).
  • 프롬프트 관리에서 AI 작성 방식을 커스터마이즈할 수 있어요.
8) 키워드 / 태그
  • 마켓 검색 시 내 상품이 잘 걸리게 하는 검색어예요.
  • AI가 자동으로 추천해줘요. 추가하거나 삭제할 수 있어요.
  • 쉼표(,)로 구분해서 입력하세요.
  • 마켓마다 태그 수 제한이 달라요.
9) 부가 정보
항목설명
원산지상품이 만들어진 나라. 중국 소싱이면 보통 "중국".
제조사 / 브랜드만든 회사 이름. 모르면 "기타" 또는 "-"로 입력하세요.
모델명상품 모델 번호. 없으면 비워두세요.
KC 인증해당하는 상품(전기제품, 어린이용품 등)은 인증 번호를 입력해야 해요.
⚠️ KC 인증 주의
전기용품, 어린이 제품 등 일부 품목은 KC 인증 없이 등록하면 마켓에서 삭제될 수 있어요.
10) 메모

마켓에는 노출되지 않는 내부 메모예요. "이 상품 잘 나가니까 재고 추가", "반품 많음 주의" 같은 내 전용 기록을 남길 수 있어요.

✅ 편집 완료 후
저장 버튼을 누르면 상태가 "수정완료"로 바뀌어요. 이제 마켓 업로드를 누르면 상품이 올라갑니다!

게시된 상품 — 마켓에 올라간 상품 관리

마켓에 올라간 상품들만 따로 모아서 볼 수 있는 화면이에요. 판매 중인 상품을 여기서 관리해요.

할 수 있는 것들
기능설명
판매 중지상품을 일시적으로 마켓에서 내려요. 재고가 없거나 잠깐 쉬고 싶을 때.
재판매판매 중지한 상품을 다시 올려요.
가격 수정마켓에 올라간 상태에서 바로 가격을 바꿀 수 있어요.
재고 수정마켓 재고 수량을 바꿔요.
삭제마켓에서 상품을 완전히 내려요. 다시 올리려면 새로 편집해야 해요.

프리셋 관리 — 자주 쓰는 설정 저장

상품마다 배송비, 마진율, 반품지를 매번 입력하기 귀찮잖아요. 한 번 저장해두면 클릭 한 번으로 끝나는 기능이에요.

프리셋 만들기
  1. 왼쪽 메뉴 → 프리셋 관리
  2. + 새 프리셋 클릭
  3. 이름 짓고 설정 입력 예: "타오바오 기본" — 마진 30%, 배송비 3,000원, 반품지 서울시 XX구
  4. 저장
프리셋 적용하기
  • 상품 1개에 적용: 편집 화면에서 프리셋 적용 드롭다운 → 원하는 프리셋 선택
  • 여러 상품에 한꺼번에 적용: 상품 목록에서 여러 개 체크 → 상단 프리셋 적용 클릭 → 프리셋 선택
일괄 프리셋 — 수십 개를 한 번에

상품 목록에서 필터로 골라낸 상품들 전체에 프리셋을 한꺼번에 적용하는 기능이에요. 100개를 올릴 때 한 번에 설정 완료!

  • 상품 목록 → 필터로 상품 선택 → 전체 선택 체크박스 → 프리셋 적용
✅ 프리셋을 적극 활용하세요
상품 카테고리별로 프리셋을 나눠두면 훨씬 빠르게 작업할 수 있어요. 예: "타오바오-패션", "알리-소형가전", "아마존-수입"

마켓 계정 설정 — 판매 채널 연결하기

상품을 올리려면 먼저 마켓 계정을 ALL8에 연결해야 해요. 마켓마다 연결 방법이 조금 달라요.

연결 방법

왼쪽 메뉴 → 마켓 계정 설정 → 연결할 마켓 선택

마켓연결 방법필요한 것
쿠팡 쿠팡 Wing(판매자 센터) API 연동 Wing 로그인 → 외부API 관리 → Access Key / Secret Key 발급 후 입력
스마트스토어 네이버 커머스 API 연동 스마트스토어 센터 → 스토어설정 → 외부앱관리 → API 키 발급 후 입력
11번가 OpenAPI 연동 11번가 OpenAPI 신청 → API 키 입력
쇼피 카카오 로그인 방식 쇼피 파트너 계정 + ALL8 연동 승인
⚠️ 연결이 안 되면
마켓에서 API 사용 신청이 필요한 경우가 있어요. 쿠팡 Wing은 바로 되는데, 스마트스토어 API는 신청 후 승인(1~2일) 후에 연결돼요.
📌 계정이 여러 개면?
같은 마켓이라도 여러 계정을 추가할 수 있어요. 계정마다 이름을 붙여두면 헷갈리지 않아요.

금지어·대체어 — 상품명 자동 정리

소싱처 상품명에 마켓에서 금지된 단어가 있으면 등록이 거부돼요. 미리 설정해두면 자동으로 걸러줘요.

금지어 — 이 단어는 삭제해줘
  • 상품명에서 특정 단어를 자동으로 삭제해요.
  • 예: "베스트셀러", "대박", "1+1" 같은 마켓 금지 표현
  • 왼쪽 메뉴 → 금지어·대체어 → 금지어 탭 → 단어 입력 후 추가
대체어 — 이 단어는 저 단어로 바꿔줘
  • 특정 단어를 다른 단어로 자동으로 교체해줘요.
  • 예: "셔츠" → "남성셔츠", "핸드폰" → "스마트폰"
  • 왼쪽 메뉴 → 금지어·대체어 → 대체어 탭 → 원래 단어 / 바꿀 단어 입력
✅ 언제 등록하면 좋아요?
처음엔 비워도 되는데, 마켓에서 등록 거부가 반복되는 단어가 생기면 그때 추가하세요. 추가하면 그다음부터 자동으로 적용돼요.

연결된 마켓에서 주문이 들어오면 ALL8에서 한 곳에 모아볼 수 있어요.

상태설명할 일
신규 주문방금 들어온 주문이에요.소싱처에서 주문하고 운송장 번호 입력.
배송 중운송장 입력 후 배송 중인 상태.고객이 받을 때까지 기다려요.

프롬프트 관리 — AI 글쓰기 방식 설정

AI가 상품명, 상세설명을 자동으로 써줄 때 어떤 스타일로 쓸지 설정하는 곳이에요. 기본값으로도 충분한데, 더 내 스타일에 맞추고 싶을 때 수정하세요.

  • 상품명 프롬프트: AI가 상품명을 만들 때 따르는 규칙. "최대 50자로", "영어 섞지 말고" 같은 지시를 추가할 수 있어요.
  • 상세설명 프롬프트: AI가 상세페이지 글을 쓸 때 따르는 규칙. "불릿 포인트로 정리", "장점 3가지 강조" 같은 방식을 설정할 수 있어요.
  • 기본값으로 되돌리기: 수정하다가 이상해지면 초기화 버튼으로 원래대로 돌릴 수 있어요.
💡 처음엔 기본값 그대로 쓰세요
기본 프롬프트도 충분히 좋아요. 몇 번 써보고 AI가 원하는 스타일로 못 쓴다고 느낄 때 수정하면 돼요.
