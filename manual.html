STEP 1 · 5분

크롬 확장 프로그램 설치하기

ALL8은 크롬 확장 프로그램으로 상품을 가져옵니다. 딱 한 번만 설치하면 돼요.

  1. all8.io 에 로그인하세요 카카오 계정으로 바로 가입·로그인할 수 있어요.
  2. 왼쪽 메뉴에서 확장프로그램 다운로드 클릭 크롬 웹스토어 페이지가 열립니다.
  3. Chrome에 추가 버튼 클릭 설치가 완료되면 브라우저 오른쪽 위에 ALL8 아이콘이 생겨요.
  4. ALL8 아이콘 클릭 → 카카오 로그인 사이드패널이 열리고 로그인하면 바로 사용할 수 있어요.
💡 크롬이 없으면?
ALL8은 크롬(Chrome) 브라우저에서만 작동해요. 엣지(Edge)나 네이버 웨일은 안 됩니다. 크롬 다운로드 →
2
STEP 2 · 2분

소싱 사이트에서 팔 상품 담기

알리익스프레스, 타오바오, 아마존, 쿠팡 등에서 팔고 싶은 상품을 ALL8으로 가져옵니다.

🛒 상품 1개 담는 법 (가장 쉬운 방법)
  1. 담고 싶은 상품 페이지를 열어요 예: 알리익스프레스에서 마음에 드는 상품 클릭
  2. 오른쪽 옆에 ALL8 사이드패널이 열려 있는지 확인 안 열려 있으면 브라우저 오른쪽 위 ALL8 아이콘을 클릭하세요.
  3. 이 상품 수집 버튼 클릭 자동으로 상품 정보를 가져오기 시작해요. 보통 10~30초 걸려요.
  4. 완료되면 all8.io 상품 목록에서 확인 상태가 "수집됨"으로 표시되면 성공이에요!
📦 여러 개를 한 번에 담고 싶다면?
상품 목록 페이지(카테고리, 검색 결과 등)에서 상품마다 체크박스가 뜹니다. 원하는 것 체크하고 수집 시작을 누르면 한꺼번에 담겨요.
🌐 사용할 수 있는 소싱 사이트
알리익스프레스 타오바오 티몰 1688 알리바바 아마존 미국/일본 쿠팡 네이버 스마트스토어 그 외 다수

지원 사이트 전체 목록은 확장 프로그램 설정(⚙) → 지원 사이트에서 확인하세요.

3
STEP 3 · 10분

편집하고 마켓에 올리기

담아온 상품을 쿠팡·스마트스토어에 올릴 수 있게 다듬는 단계예요. 어렵지 않아요 — 위에서 아래로 채우면 됩니다.

  1. all8.io → 상품 목록에서 담아온 상품 클릭 편집 화면으로 이동해요.
  2. 마켓 선택 — 어디에 올릴지 고르기 쿠팡, 스마트스토어, 11번가 등 미리 연결해둔 마켓을 선택하세요.
  3. 상품명 확인 · 수정 AI가 자동으로 번역·생성해줘요. 마음에 안 들면 수정하세요.
  4. 카테고리 선택 마켓에서 어떤 카테고리에 올릴지 선택해요. AI가 추천해줘서 대부분 그대로 써도 돼요.
  5. 가격 설정 원가, 마진, 판매가를 입력하면 자동으로 계산해줘요.
  6. 이미지 확인 자동으로 가져온 이미지를 확인하고, 순서를 바꾸거나 불필요한 건 제거하세요.
  7. 상세 설명 확인 AI가 자동 생성해줘요. 필요하면 수정하세요.
  8. 마켓 업로드 클릭! 몇 초~몇 분 안에 마켓에 등록됩니다. 상태가 "판매중"이 되면 성공!
🎉 축하해요!

에러가 뜨면? 대부분 마켓 계정 연결 문제예요. 아래 FAQ를 확인하세요.

자주 묻는 질문
all8.io 왼쪽 메뉴 → 마켓 계정 설정 → 연결할 마켓 선택 → 아이디/비밀번호 입력하면 돼요. 쿠팡은 Wing 계정 정보를 입력하고, 스마트스토어는 API 키를 발급받아 넣어요. 연결이 안 되면 단톡방에서 물어보세요!
상품을 담는 방식이 2가지예요.

브라우저 수집 (기본, 무료) — 내 컴퓨터의 크롬이 직접 상품 페이지를 열어서 정보를 가져와요. 그래서 컴퓨터가 켜져 있어야 해요.

서버 수집 (크레딧 소모) — ALL8 서버가 대신 가져다줘요. 컴퓨터가 꺼져 있어도 되고 더 빠른데, 쓸 때마다 크레딧이 조금 차감돼요.

처음엔 그냥 브라우저 수집만 써도 충분해요.
서버 수집이나 일부 AI 기능을 쓸 때 소모되는 포인트예요. 기본 브라우저 수집은 크레딧이 필요 없어요. 크레딧은 all8.io 결제 메뉴에서 충전할 수 있어요.
"배송비 3,000원, 마진 30%, 반품지 서울시" 같은 자주 쓰는 설정을 저장해두는 기능이에요.

한 번 만들어두면 상품 편집할 때 프리셋 적용 한 번으로 설정이 한꺼번에 들어가요. 상품 100개를 올릴 때 매번 같은 걸 입력 안 해도 되니까 꼭 쓰세요!

왼쪽 메뉴 → 프리셋 관리에서 만들 수 있어요.
소싱처에서 상품이 품절되면 내 마켓에도 빨리 반영해야 해요. 안 그러면 고객이 주문했는데 정작 못 보내는 상황이 생겨요.

재고연동을 켜두면 ALL8이 주기적으로 소싱처를 확인해서 품절 여부를 알려줘요. 상품이 어느 정도 쌓이면 꼭 켜두세요.
1. 마켓 계정 연결 확인 — 가장 흔한 원인이에요. 마켓 계정 설정에서 연결 상태를 확인하세요.
2. 재시도 버튼 클릭 — 상품 상세 패널에서 에러 옆 재시도를 눌러보세요.
3. 그래도 안 되면 — 단톡방에 에러 화면 캡처해서 올려주세요. 바로 도와드려요!

👉 단톡 무료교육 듣기
방법 1 — 체크박스로 여러 개 선택
상품 목록에서 여러 상품 체크 → 프리셋 적용 → 마켓 업로드 순서로 한꺼번에 처리할 수 있어요.

방법 2 — 엑셀로 URL 대량 업로드
확장 프로그램 → + URL · 엑셀로 추가에서 URL 목록을 붙여넣거나 엑셀 파일로 올리면 한 번에 수백 개도 담을 수 있어요.

📋 상품 목록 화면

수집한 상품을 한 곳에서 관리해요. 여러 개 선택해서 한꺼번에 처리할 수 있어요.

📥 수집됨 ⚙️ 처리중 ✏️ 편집 중 ✅ 수정완료 ⬆️ 업로드 중 🟢 판매중 ❌ 에러
🔧 프리셋 — 시간 절약의 핵심

자주 쓰는 설정(배송비, 마진, 반품지 등)을 저장해두는 기능이에요. 상품 100개 올릴 때 매번 같은 설정 입력 안 해도 돼요. 왼쪽 메뉴 → 프리셋 관리에서 만드세요.

🔄 재고연동 — 품절 자동 감지

소싱처 상품이 품절되면 자동으로 알려줘요. 상품 목록에서 여러 개 선택 → 재고 연동 설정으로 켤 수 있어요. 상품이 쌓이면 꼭 켜두세요.

🏪 게시된 상품 — 판매 중인 상품 관리

마켓에 올라간 상품만 모아서 보는 화면이에요. 판매중지, 재판매, 가격 수정 등을 여기서 할 수 있어요.

🚫 금지어 · 대체어

상품명에서 특정 단어를 자동으로 삭제하거나 다른 단어로 바꿔주는 기능이에요. 예: "짝퉁" 같은 단어가 포함되면 마켓 등록이 거절되니 미리 필터링할 수 있어요. 왼쪽 메뉴 → 금지어·대체어에서 설정하세요.
