ALL8 사용 설명서

ALL8은 소싱 사이트에서 상품을 가져와 쿠팡·스마트스토어 같은 마켓에 자동으로 등록해주는 프로그램이에요. 이 설명서는 모든 메뉴를 빠짐없이 상세하게 다룹니다 — 화면에 보이는 버튼 하나하나, 각 선택지의 정확한 의미까지 담았어요. 원하는 메뉴를 왼쪽에서 고르거나 위 검색창으로 찾아보세요.

화면 맨 위에 노란색 공지 바가 보일 때가 있어요 — 운영팀이 올린 중요 안내이니 한 번 확인하고 X 버튼으로 닫으면 돼요.

대시보드

로그인하면 가장 먼저 보이는 화면이에요. 지금 내가 등록한 상품들이 어떤 상태에 몇 개씩 있는지 한눈에 확인할 수 있어요.

상태 카드 7개

카드의미
수집됨확장 프로그램으로 담아왔지만 아직 편집을 시작하지 않은 상품
처리중수집한 상품 정보를 서버가 정리하고 있는 중
편집 중편집을 시작했지만 아직 다 마치지 않은 상품
수정완료편집을 마쳐서 마켓에 올릴 준비가 된 상품
업로드 중지금 마켓에 등록되고 있는 중
판매중마켓에 정상적으로 등록되어 팔리고 있는 상품
에러마켓 등록에 실패한 상품

카드를 클릭하면 상품 목록 화면으로 이동해요. 특정 상태의 상품만 보고 싶다면, 이동한 뒤 목록 화면 위쪽의 상태 탭을 한 번 더 눌러 필터링해주세요.

아래쪽에는 최근에 바뀐 상품들이 나열돼요. 클릭하면 바로 그 상품의 편집 화면으로 이동합니다. 그 아래 소싱 상품 · 편집 상품 · 프리셋 관리 버튼으로 자주 가는 화면에 빠르게 이동할 수 있어요.

상품 목록

내가 수집한 상품과 편집 중인 상품을 한 곳에서 모아 관리하는 화면이에요. 여러 상품을 한 번에 선택해서 프리셋을 적용하거나 마켓에 업로드하는 등 다양한 일괄 작업을 할 수 있어요.

  1. 위쪽 상태 탭으로 원하는 상태의 상품만 골라 볼 수 있어요 (전체 / 수집됨 / 처리중 / 편집 중 / 수정완료 / 업로드 중 / 판매중 / 에러).
  2. 검색창에 상품명을 입력하면 자동으로 찾아줘요.
  3. 상품 체크박스를 눌러 여러 개를 선택하면 화면 아래에 일괄 작업 버튼들이 나타나요.

여러 개 선택했을 때 나오는 버튼들

  • 프리셋 적용 — 선택한 상품들에 미리 만들어둔 프리셋을 한 번에 적용해요.
  • 엑셀로 추출 — 선택한 상품 정보를 엑셀 파일로 내려받아요.
  • 마켓 업로드 — 선택한 상품들을 연결된 마켓 계정으로 바로 등록해요.
  • 재고 연동 설정 — 선택한 상품들의 소싱 원본 재고(품절 여부)를 주기적으로 자동 확인하도록 켜요. 자세한 내용은 아래 "재고연동" 항목을 참고하세요.
  • 복사/이동 — 상품을 다른 워크스페이스나 목록으로 복사하거나 옮겨요. 완료되면 화면 아래에 결과를 알려주는 작은 알림(토스트)이 잠깐 떴다가 사라지고, 필요 없으면 눌러서 바로 닫을 수 있어요.
  • 혼합구성 만들기 — 여러 상품(옵션)을 묶어서 하나의 세트 상품으로 만들어요. 자세한 사용법은 아래를 참고하세요.
  • 삭제 — 선택한 상품을 목록에서 삭제해요.

상품을 클릭하면 편집 화면으로 이동해요. 아직 편집을 시작하지 않은 상품이면 자동으로 편집 준비가 된 뒤 이동합니다.

참고 상단의 수동 등록 버튼은 아직 준비 중인 기능이에요. 수집 처리 중 오류가 나면 편집 화면에 "소싱처 열기" 버튼이 함께 떠서, 원래 소싱 페이지를 바로 열어 다시 확인해볼 수 있어요.

혼합구성 만들기, 자세히

여러 상품(또는 한 상품의 여러 옵션)을 하나로 묶어 "세트 상품"으로 만드는 기능이에요. 예를 들어 "그릇 세트"를 만들 때 색상마다 다른 구성품을 담고 싶을 때 유용해요. 상품 목록에서 상품을 선택하고 혼합구성 만들기를 누르면 전용 편집 창이 열립니다.

  1. 창 위쪽에서 세트의 옵션값(예: 색상별 구성)을 추가해요 — 최대 100개까지 만들 수 있어요. 왼쪽 레일(목차)로 옵션값 사이를 빠르게 오갈 수 있고, 옵션값이 6개를 넘어가면 카드가 자동으로 요약되어 화면이 복잡해지지 않아요.
  2. 각 옵션값마다 어떤 상품(SKU)을 몇 개씩 담을지 고르고, 수량은 +/− 버튼(스테퍼)으로 조절해요. 옵션값별로 대표 이미지와 "구성 상세"(세트 안에 뭐가 들었는지 설명)도 따로 채울 수 있어요.
  3. 카드 왼쪽의 손잡이(⠿) 아이콘을 드래그하면 옵션값 순서를 바꿀 수 있어요.
  4. 화면 아래에 "최대 N세트"가 실시간으로 표시돼요 — 담은 상품들의 실제 재고를 기준으로 몇 세트까지 만들 수 있는지 자동 계산한 값이에요. 재고가 0인 구성이 섞여 있으면 빨간 글씨로 경고가 떠요.
  5. 아직 다 채우지 못한 항목이 있으면 하단 안내 문구가 어느 섹션이 비어있는지 알려줘요. 다 채운 뒤 저장하면 완료.
이미지 역할 이름이 여기서는 조금 달라요 이 창 안의 이미지 역할 이름은 "대표 이미지(썸네일)" → "상품 이미지(보조 썸네일)" → "상세설명 이미지" 순서로 표시돼요. 이건 혼합구성 만들기 창 안에서만 쓰이는 이름이고, 일반 상품 편집 화면의 "이미지" 섹션 이름과는 별개예요.

게시된 상품

이미 마켓에 실제로 등록(업로드)된 상품만 모아서, 판매 성과를 확인하고 판매 상태를 관리하는 화면이에요. "상품 목록"이 수집·편집 단계 상품을 다룬다면, 이 화면은 이미 팔리고 있는 상품을 관리하는 곳이라고 생각하면 됩니다.

화면은 세 부분으로 나뉘어요: 왼쪽은 필터, 가운데는 상품 목록, 오른쪽은 클릭한 상품의 상세 정보(인스펙터)예요.

왼쪽 — 필터 및 그룹

  • 작업 큐 — 확인 필요 / 업데이트 필요 / 처리중 / 비활성 / 부분 판매 / 노출 없음 / 고도화 / 관찰 / 유지 상태로 필터링
  • 마켓 / 계정 / 사업자 — 등록해둔 마켓 계정·사업자 기준으로 필터링
  • 판매상태 — 판매중 / 판매중지 / 품절 / 삭제
  • 태그 / 내 그룹 — 직접 붙인 태그나, 원하는 상품끼리 묶어둔 나만의 그룹
  • 저장된 뷰 — 자주 쓰는 필터 조합을 "현재 조합 저장"으로 북마크해두고 나중에 바로 불러올 수 있어요

같은 항목을 여러 개 선택하면 "또는"(하나라도 해당)/"그리고"(전부 해당) 중 어떻게 조합할지도 고를 수 있어요.

가운데 — 상품 목록

썸네일, 상품명, 판매가, 마켓별 판매상태(마켓마다 색깔 점으로 표시 — 초록: 판매중 / 노랑: 중지·품절·진행중 / 회색: 종료 / 빨강: 거절·에러), 최근 방문·판매 추이 등이 표로 보여요. 정렬은 최근 업데이트순 / 조회순 / 조회 급상승순 / 마켓 수 많은순 / 상품명순 중 고를 수 있고, 검색창에서 상품명·상품번호·계정명으로 찾을 수 있어요. 메모·태그 칸을 클릭하면 바로 수정할 수 있는 창이 떠요.

오른쪽 — 상품 클릭 시 열리는 상세 패널

상품을 클릭하면 오른쪽에 그 상품의 마켓별 등록 현황이 카드로 나열돼요. 각 카드에는 실제 판매 페이지 링크, 조회수, 예상 마진, 오류가 있으면 사유와 재시도 버튼이 함께 표시됩니다. 상단에 버튼 3개가 있어요:

  • 판매상태 동기화 — 마켓에 실제로 등록된 판매상태를 다시 읽어와서 최신화해요.
  • 편집 — 상품명·가격·이미지 같은 내용을 고치고 싶으면 이 버튼으로 편집 화면으로 이동해요. 이 "게시된 상품" 화면 자체에서는 가격·재고를 직접 고칠 수 없어요 — 반드시 편집 화면으로 이동해야 합니다.
  • 삭제 — 아래 "삭제 기능, 헷갈리지 않게" 참고.

카드 하단의 판매중/판매중지/품절/삭제 버튼으로 마켓별 판매 상태를 개별로 바꿀 수도 있고, 목록에서 여러 상품을 선택하면 같은 작업(판매상태 변경, 태그 추가/제거, 그룹 담기, 업데이트)을 한꺼번에 할 수도 있어요.

삭제 기능, 헷갈리지 않게 목록에서 여러 개 선택 후 누르는 "삭제"는 마켓의 판매 상태를 "판매종료"로 바꾸는 것뿐이고, 상품 데이터 자체는 남아있어요.
상세 패널(인스펙터) 안의 "삭제" 버튼은 완전히 달라요 — 판매 중인 모든 마켓 상품을 먼저 판매종료(11번가는 판매중지) 처리한 뒤, 전부 성공하면 수집 상품 데이터와 저장된 이미지까지 통째로 삭제합니다. 확인창에 "판매중인 모든 마켓 상품을 삭제합니다... 삭제하시겠습니까?"라고 뜨니 신중하게 눌러주세요. 아직 마켓 처리가 끝나지 않은 상품이 섞여 있으면 삭제가 진행되지 않고 안내 문구가 떠요.

상품 수집 — 기본 사용법

알리익스프레스, 타오바오, 1688, 아마존, 쿠팡 등 소싱 사이트에서 상품 정보를 자동으로 가져와 ALL8에 등록하는 기능이에요. 크롬 확장 프로그램을 설치하면 브라우저 옆에 작은 창(사이드패널)이 뜨는데, 여기서 상품을 담으면 됩니다.

아직 확장 프로그램이 없다면 사이드바의 "확장프로그램 다운로드" 메뉴에서 설치할 수 있어요.

처음 시작할 때 — 로그인

확장 프로그램 아이콘을 누르면 사이드패널이 바로 열려요. 처음엔 카카오 로그인 버튼만 보이는데, 눌러서 카카오 계정으로 로그인하면 바로 사용할 수 있어요. 로그인 중에는 버튼 문구가 확인 중…으로 바뀌어요.

화면 구성 — 탭 3개 + 설정

  • 수집 — 기본 화면. 상품 URL이나 목록 페이지에서 상품을 담아 자동으로 수집해요.
  • 수동수집 — 자동 수집이 안 되는 사이트에서 이미지와 정보를 직접 입력해 등록해요.
  • 재고연동 — 이미 등록한 상품이 품절됐는지 주기적으로 확인해주는 기능이에요. (워크스페이스 소유자에게만 이 탭이 보여요)
  • 오른쪽 위 ⚙ 설정 — 지원 사이트 목록(사이트/수집방식/재고연동/상태)과 중복수집 허용 토글이 있어요.

수집·수동수집 탭 위쪽에는 항상 "보낼 워크스페이스" 드롭다운이 있어, 수집한 상품을 어느 작업공간으로 보낼지 매번 확인할 수 있어요.

상품을 담는 방법 3가지

  1. 상품 상세 페이지에서 바로 담기 — 지원하는 사이트의 상품 페이지를 열어두고 사이드패널을 보면, 상황에 따라 버튼 문구가 달라져요: 큐가 비어있으면 이 상품 수집(누르면 즉시 담기고 수집 시작), 큐에 이미 대기 항목이 있으면 ＋ 대기열에 추가, 이미 수집이 진행 중이면 진행 중 수집에 합류.
  2. 목록 페이지에서 여러 개 한 번에 담기 — 상품 목록이나 카테고리 페이지를 열면, 지원 사이트는 상품마다 체크박스가 자동으로 나타나요. 원하는 상품을 체크하거나 페이지의 모든 상품 선택으로 한 번에 담을 수 있어요.
  3. URL을 직접 붙여넣거나 엑셀로 올리기 — ＋ URL · 엑셀로 추가를 눌러 URL 목록(줄바꿈/콤마/공백으로 구분)을 붙여넣거나, 엑셀 양식 다운로드로 받은 양식(url 주소 / 프리셋 ID / 옵션 구성 수집 3열, 최대 2MB·2만행)을 채워 엑셀/CSV 업로드할 수 있어요.

처음 방문하는 사이트에서는 크롬이 접근 권한을 물어봐요 — 허용을 눌러야 수집이 진행돼요.

수집 진행 상황 확인

대기열 목록 헤더에 수집 대기열 (N)과 완료율(%)이 표시돼요. 툴바에는 수집 시작 (N) / ■ 중지 / ↻ 실패 N(일괄 재시도) / 완료·실패 정리 / 전체 비우기 버튼이 있어요. 카드별 상태는 대기 / 진행중 / 실패 / 완료 중 하나로 표시돼요.

실패한 카드는 옆의 ↻ 아이콘으로 개별 재시도할 수 있어요(단, "중복 상품"으로 실패한 건 재시도 버튼이 안 뜨는데, 이미 수집된 상품이라 재시도할 필요가 없기 때문이에요). 여러 개가 한꺼번에 실패했다면 ↻ 실패 N으로 한 번에 재시도할 수 있어요.

알아두면 좋아요 실패 사유는 대부분 친절한 한국어 문구로 뜨지만(권한 거부, 중복 상품 등), 드물게 "navigate-fail"처럼 영어로 된 사유가 뜰 수 있어요. 이럴 땐 그냥 재시도 버튼을 눌러보고, 계속 안 되면 문의해주세요.

수동수집

자동 수집을 지원하지 않는 사이트에서 상품을 등록할 때 쓰는 기능이에요. 페이지의 이미지를 가져와서 상품명·옵션·가격을 직접 입력하면 그대로 서버에 등록됩니다. 번역이나 마진 계산 없이 입력한 그대로 전송돼요.

  1. "수동수집" 탭을 열면 자동으로 현재 페이지의 이미지를 불러오려고 시도해요. 처음 방문하는 사이트면 권한 허용하고 이미지 읽기 버튼을 눌러 접근을 허용해야 해요. (http/https 페이지가 아니면 이미지를 수집할 수 없다는 안내가 떠요)
  2. 페이지 이미지 카드에서 원하는 이미지를 클릭해 선택해요. 선택하면 파란 테두리로 표시되고, 그 아래 썸네일 / 상세 버튼으로 역할을 정할 수 있어요. 선택하는 순간 기본은 "상세"로 지정되고, 필요하면 "썸네일"로 바꿔주면 돼요.
  3. 기본 정보 카드에 상품명(필수), 상품 설명(선택), 브랜드(선택), 제조사(선택), 배송비(선택, 원본 그대로)를 입력해요.
  4. 옵션 카드에서 + 옵션그룹으로 색상·사이즈 같은 옵션을 추가하고, 값은 입력 후 Enter로 하나씩 추가해요.
  5. 가격 카드에서 통화(원화/달러/엔화/위안화/유로/파운드 중 선택, 기본은 원화)를 고르고, 옵션 조합별로 원가격과 재고 수량을 입력해요. 각 옵션 행마다 판매중/품절 토글도 있어요(기본은 판매중).
  6. 모든 옵션에 원가격이 입력되어야 확인 (서버로 전송) 버튼이 눌려요. 전송되면 "상품을 서버로 전송했습니다" 안내와 함께 화면이 초기화돼요.
참고 재고 수량 칸을 비워두면 수량 없이 판매 상태(판매중/품절)만 전송돼요. 한글로 옵션값을 입력할 때 조합 중간에 Enter를 눌러도 중복으로 추가되지 않도록 처리되어 있어요.

재고연동

이미 등록한 상품이 소싱처에서 품절되지는 않았는지 주기적으로 자동 확인해주는 기능이에요. 확인 주기, 횟수, 확인 방식을 원하는 대로 설정할 수 있어요.

재고연동 켜기

  1. 웹사이트의 상품 목록 화면에서 재고 연동을 켜고 싶은 상품을 체크박스로 선택해요.
  2. 하단에 나타나는 재고 연동 설정 버튼을 눌러요.
  3. 안내창: "선택한 상품의 원본 판매상태와 옵션 품절을 주기적으로 확인합니다. 실제 확인은 브라우저 익스텐션이 켜져 있을 때 수행돼요." — 연동 켜기 (N개)를 누르면 완료.

단, 수동으로 직접 등록한 상품(소싱 원본이 없는 상품)이나 재고연동을 지원하지 않는 소싱처의 상품은 켤 수 없고, 안내 문구로 제외 사유가 표시돼요.

확인 계획 설정 (주기 · 횟수 · 방식)

  • 확인 주기 — 숫자 + "시간마다"/"일마다" (기본 6시간마다)
  • 총 확인 횟수 — 숫자 + "회" (기본 10회), "무제한" 체크 가능
  • 재고 확인 방식:
    • 복합(기본값) — 브라우저로 먼저 확인하고, 실패하면 서버로 자동 전환
    • 브라우저 — 로그인이 필요한 사이트도 있고, 너무 자주 확인하면 계정이 차단될 가능성도 있어요.
    • 서버 — 크레딧이 차감돼요. 가격이 원본과 조금 다르게 나올 수 있어요.

이미 켜둔 상품의 계획을 바꾸려면, 상품 목록 표의 "재고연동" 칸(파란 글씨)을 클릭하면 됩니다. 주의: 계획을 수정해서 저장하면 값이 그대로여도 "확인 횟수"가 0부터 다시 시작돼요.

개별 "재고확인" 버튼은 예외예요 상품 하나하나 옆의 재고확인 버튼을 직접 누르면, 설정된 방식(브라우저/서버/복합)과 상관없이 항상 서버로 확인하고, 확인 횟수를 깎지 않아요. 급하게 지금 당장 확인하고 싶을 때 쓰면 됩니다.

  • 품절된 옵션만 재고 0 — 품절된 옵션만 재고를 0으로 바꾸고, 나중에 재입고가 확인되면 자동으로 재고가 복원돼요.

ESM(G마켓/옥션)처럼 그룹으로 묶어 등록하는 마켓에 업로드할 때는, 마켓에서 처리가 끝날 때까지 시간이 걸릴 수 있어서 버튼이 마켓 처리 중…으로 바뀌고 최대 5분까지 기다려요. 5분이 지나도 끝나지 않으면 안내 문구가 뜨는데, 이땐 잠시 후 "게시된 상품" 화면에서 실제로 등록됐는지 확인해보면 돼요.

화면 위쪽 "동작 ▾" 드롭다운에서 상품 복제를 누르면, 지금 이 상품을 통째로 복사한 새 상품을 만들어요(별도 확인창 없이 바로 저장 → 복제 → 상품 목록으로 이동합니다). 예전에 있던 "상품 ID 복사" 메뉴는 없어졌는데, 상품번호는 이제 화면 상단의 번호를 클릭하면 바로 복사돼요(아래 참고 참조).

참고 화면 상단의 상품번호를 클릭하면 클립보드에 복사돼요("복사됨" 표시가 잠깐 뜹니다). 문의하실 때 이 번호를 함께 보내주시면 빠르게 확인할 수 있어요. 같은 방식으로 프로필 메뉴에서도 내 계정 ID를 복사할 수 있어요.

프리셋 섹션

미리 만들어둔 프리셋을 이 상품에 바로 적용할 수 있는 곳이에요. 카드를 클릭하면 즉시 상품명·마켓·옵션 등 관련 항목들이 그 프리셋대로 채워집니다. 화면 위쪽의 저장 버튼을 눌러야 실제로 저장돼요. (프리셋 자체를 만드는 방법은 아래 "프리셋 관리" 항목을 참고하세요)

마켓 선택

이 상품을 어느 마켓 계정에 올릴지 고르는 곳이에요. 이미 등록해둔 마켓 계정(마켓 계정 설정 메뉴에서 만든 것)들이 사업자별로 묶여서 나타납니다.

  1. 화면에는 사업자(비즈니스)별로 그룹이 나열돼요. 사업자 이름이 없으면 "기본 사업자"로 묶여요.
  2. 각 그룹 위에 전체 선택 버튼이 있어요 — 누르면 그 사업자의 활성화된 계정 전체가 선택되고, 다시 누르면 전체 해제돼요. 일부만 선택돼 있으면 체크박스가 반쯤 채워진 모양으로 표시돼요.
  3. 그 아래 마켓 계정 카드들을 클릭해서 개별로 선택/해제할 수 있어요. 카드에는 마켓 배지, 계정 이름, 마켓 이름이 표시되고, 비활성화된 계정은 흐리게 표시되며 클릭할 수 없어요.

카드에 표시되는 업로드 상태

  • 업로드됨(초록) — 이미 이 마켓에 정상 등록됨, 등록 시각도 함께 표시
  • 수정 중(주황) — 수정한 내용이 마켓에 반영되고 있는 중
  • 실패 시 카드 안에 빨간 박스로 실패 사유가 표시돼요

상품명

마켓에 노출될 상품명을 정하는 곳이에요. 원본 그대로 쓸 수도 있고, AI로 자연스럽게 다시 만들 수도 있어요.

  1. 기본 상품명 입력창에 직접 타이핑해서 수정할 수 있어요.
  2. AI로 새로 만들고 싶으면 ↻ 재번역을 눌러 패널을 펼치고, 모델과 프롬프트를 고른 뒤 상품명 생성을 눌러요. 항상 소싱 원본 상품명을 기준으로 새로 만들어져요(지금 입력해둔 이름이 아니라).
  3. 그 아래 원본 상품명은 참고용으로만 보여지는 읽기 전용 텍스트예요 (소싱 사이트에서 가져온 원문).
  4. "▾ 마켓별 상품명"을 펼치면, 선택한 마켓 계정마다 다른 상품명을 따로 입력할 수 있어요. 비워두면 위의 기본 상품명이 그대로 쓰여요. 마켓별로도 각각 AI 재번역이 가능해요.
참고 상품명 글자수 제한을 넘기거나 금지어가 포함되면 빨간 경고 문구가 바로 뜹니다.

카테고리

이 상품이 각 마켓에서 어느 카테고리로 등록될지 정하는 곳이에요. 마켓마다 카테고리 체계가 달라서, 마켓별로 한 줄씩 따로 지정합니다.

  1. 화면을 열면 각 마켓 줄에 AI가 추천한 카테고리가 회색 글씨로 미리 채워져 있어요(예: "패션의류 > 여성의류 > 반팔티셔츠 (AI 추천)"). 그대로 둬도 괜찮으면 별도 조작이 필요 없어요.
  2. 직접 바꾸고 싶으면 그 줄을 클릭 → 검색창에 원하는 키워드 입력 → 후보 목록이 뜨면 선택해요.
  3. 후보를 고를 때 이 마켓만(그 마켓 하나에만 적용) 또는 전 마켓(다른 마켓들도 비슷한 카테고리로 자동 매칭 시도 → 미리보기 확인 후 적용) 중 고를 수 있어요. "전 마켓"을 누르면 마켓별 매칭 후보와 일치율(%)이 표시되는 확인창이 뜨고, 적용을 눌러야 실제로 반영돼요. 자동 매칭에 실패한 마켓은 별도로 수동 선택해야 해요.
  4. 바꾼 걸 되돌리고 싶으면 되돌리기 버튼을 눌러요.
카테고리별 필수 정보는 여기 없어요 인증정보, 원산지 같은 카테고리별 필수 입력 항목은 이 섹션이 아니라 아래 "부가 정보" 섹션의 "마켓별 조정"에서 입력해요.
ESM(G마켓/옥션) "노옵션" 배너 ESM 카테고리 중에는 옵션(색상·사이즈 등)을 아예 등록할 수 없거나, 정해진 값만 허용하는 카테고리가 있어요. 이런 카테고리를 고르면 화면에 "노옵션" 안내 배너가 함께 뜨는데, 실제로 이 옵션 정책을 마켓 계정 단위로 어떻게 처리할지는 "마켓 계정 설정" 화면에서 정해요. 자세한 내용은 아래 "마켓 계정 설정" 항목을 참고하세요.
알아두면 좋아요 카테고리를 잘못 골랐는지는 이 화면에서 미리 알려주지 않아요 — 실제로 업로드를 시도했을 때에만 문제(예: "이 카테고리는 이 마켓에 등록할 수 없습니다")를 알 수 있어요. 카테고리 관련 오류가 뜨면 이 섹션으로 돌아와 더 구체적이거나 다른 카테고리로 바꿔보세요.

옵션 및 가격

상품의 판매 가격을 정하고, 색상·사이즈 같은 옵션별로 재고와 가격을 관리하는 곳이에요. 원가에 마진을 붙여 자동으로 판매가를 계산해주는 계산기가 들어 있어서, 매번 직접 계산할 필요가 없어요.

가격 계산기 사용법

  1. "계산식 설정" 부분을 누르면 계산기 창이 열려요.
  2. 비용(매입가, 환율, 카드수수료, 관부가세, 부가세 등)과 마진(곱하기마진, 더하기마진) 값을 입력해요.
  3. 계산식은 직접 타이핑하지 않고, 변수 버튼과 연산 기호 버튼(+ − × ÷)을 눌러서 조립해요. 예: 매입가 × 곱하기마진
  4. 값을 입력하고 나면 자동으로 모든 옵션의 판매가가 다시 계산돼요. 따로 "적용" 버튼을 누를 필요가 없어요.
  5. 매입가 구간마다 마진을 다르게 주고 싶다면 "구간별 계산식"에서 설정할 수 있어요.
  6. 단수처리(예: 100원 단위로 올림)도 여기서 정할 수 있어요.

옵션별 가격표(SKU 표)

색상·사이즈 등 옵션 조합마다 한 줄씩 나열되고, 판매가와 재고를 직접 수정할 수도 있어요. 특정 옵션 하나만 가격을 다르게 하고 싶다면 그 칸을 직접 고치면 되는데, 이후에 계산기 값을 다시 바꾸면 그 값도 함께 재계산될 수 있어요. 그럴 땐 그 줄에 나타나는 계산식 조정 버튼을 눌러서, 원하는 가격에 맞게 계산식 자체를 맞춰줄 수 있어요.

각 옵션 줄의 매입가 칸도 직접 고칠 수 있어요. 고치고 나서 원래 값으로 되돌리고 싶으면 옆의 되돌리기 아이콘을 누르면 돼요. 모델번호·바코드 같은 식별자는 옵션마다 하나씩 입력할 수도 있고, 여러 옵션에 한꺼번에 같은 규칙으로 채워 넣는 일괄 입력 기능도 있어요.

이미 마켓에 업로드된 옵션은 삭제할 수 없어요 — 삭제 버튼 대신 품절 처리 버튼이 나타나요. 이제 안 파는 옵션이라면 삭제 대신 품절 처리를 눌러 재고를 0으로 만들어주세요(마켓에 이미 등록된 옵션 정보 자체는 남아있어야 판매 이력 등이 꼬이지 않기 때문이에요).

할인율(%) — 화면에만 보이는 "할인처럼 보이는" 표시

가격 계산기 근처에 할인율(%) 입력란이 있어요. 여기에 숫자를 넣으면, 실제 판매가는 그대로 두고 마켓 화면에 정가(취소선)와 할인된 것처럼 보이는 가격을 함께 보여줄 수 있어요. 예를 들어 판매가를 9,000원으로 정하고 할인율을 10으로 넣으면, 마켓에는 정가 10,000원에 취소선이 그어지고 실제 결제금액은 그대로 9,000원으로 표시돼요. 실제로 받는 돈(판매가)이 바뀌는 기능이 아니라는 점을 꼭 기억해주세요 — 순수하게 화면 표시용이에요.

마켓별로 가격을 다르게 하고 싶을 때

마켓마다 수수료가 다르기 때문에, 마켓 계정별로 판매가를 따로 조정할 수 있는 표가 아래에 있어요. 비워두면 기본 판매가에 그 마켓 수수료만 자동으로 반영돼서 계산돼요.

알아두면 좋아요 업로드하려면 재고가 1개 이상인 옵션이 최소 하나는 있어야 해요. 모든 옵션의 재고가 0이면 "재고가 있는 판매 옵션이 없습니다" 안내와 함께 업로드가 막히니, 이 화면에서 재고를 먼저 채워주세요. 일부 옵션만 재고 0이면 그 옵션만 빠지고 나머지는 정상적으로 올라가요.

배송 및 반품

이 상품의 배송비와 반품 관련 정책을 설정하는 곳이에요.

배송비 유형 4가지

유형추가로 나타나는 입력 필드
무료반품 배송비만 입력
고정판매배송비 → 반품 배송비
조건부무료배송 기준금액 → 판매배송비 → 반품 배송비

반품 배송비는 편도 배송비이고, 이 배송비 유형과 상관없이 항상 입력할 수 있어요.

11번가는 예외예요 11번가는 "조건부" 유형을 지원하지 않아서, 등록할 때 자동으로 "고정"으로 바뀌고 무료배송 기준금액은 무시돼요. 또 고정 배송비가 판매가의 50%를 넘으면 등록 자체가 거부되니 참고하세요.

교환 배송비(왕복)

이건 "배송 및 반품" 섹션이 아니라 "부가 정보" 섹션에서(스마트스토어·11번가·토스 계정에서만) 설정해요. 별도로 입력하지 않으면 "반품 배송비 × 2"로 자동 계산되고, 필요하면 직접 값을 고칠 수도 있어요.

택배사·발송마감·도서산간 추가비는 여기 없어요

이런 항목들은 상품마다 입력하는 게 아니라 마켓 계정 설정에서 계정당 한 번만 설정합니다(예: 쿠팡의 도서산간 배송 가능 여부, 롯데온의 평일 발송 마감 시간, 11번가/토스의 제주·도서산간 추가 배송비 등).

마켓별로 배송 설정을 다르게 하고 싶을 때

섹션 하단의 "마켓별 배송 설정"에서 마켓 계정별로 배송 정보를 따로 지정할 수 있어요. 실제 적용되는 값은 다음 우선순위로 정해져요: ① 지금 이 화면에서 마켓별로 새로 입력한 값 → ② 예전에 저장해둔 마켓별 설정값 → ③ 마켓 계정 설정의 기본값 → ④ 이 섹션(공통) 설정값. 화면에 지금 어떤 값이 적용되고 있는지 표시되니 참고하세요.

이미지

상품 이미지를 추가·정리하고, AI로 배경을 지우거나 이미지 속 외국어 글자를 한글로 바꿔주는 기능이에요.

  1. 이미지 카드를 클릭하거나 체크박스로 한 장 이상 선택하면 위쪽에 처리 버튼들이 나타나요.
  2. 배경제거 — 이미지 배경을 지워줘요.
  3. 텍스트번역 — 이미지 안에 있는 외국어 글자를 한글로 바꿔줘요.
  4. 텍스트제거 — 이미지 안의 글자를 지워줘요.
  5. 처리 결과를 새 이미지로 추가할지, 원래 이미지를 대체할지 선택할 수 있어요.
  6. 처리가 끝나면 자동으로 이미지가 바뀐 걸 확인할 수 있어요.

이미지 추가는 + 이미지 추가를 눌러 URL 입력, 파일 업로드, 또는 이미지 보관함에서 고를 수 있어요. 순서 변경은 카드를 드래그해서 옮기면 되고, 대표 이미지 자리에 끌어다 놓으면 그 이미지가 대표 이미지가 돼요. 삭제는 여러 장을 선택한 뒤 한 번에 지울 수 있어요.

참고 회전·좌우반전 버튼은 아직 준비 중이라 눌러도 이미지가 바뀌지 않아요.

옵션(색상·사이즈 등)에 달린 이미지는 이 화면이 아니라 "옵션 및 가격" 화면의 옵션 카드에서 처리해요(회전·좌우반전·배경제거·텍스트번역·텍스트제거 버튼이 있어요). 상세 설명에 들어갈 이미지 관련 기능은 다음 항목("상세 설명")에서 다룹니다.

  4. 스마트스토어 계정이 있으면 네이버 키워드 추천 버튼도 함께 있어서, 상품명 기준으로 추천 키워드를 받아와 추가할 수 있어요.

일괄 프리셋 적용

여러 상품을 하나하나 고치지 않고, 미리 만들어둔 설정(프리셋)을 여러 상품에 한꺼번에 적용하는 기능이에요. 상품 100개에 똑같은 상품명 규칙이나 가격 공식을 적용하고 싶을 때 유용해요.

  1. 상품 목록에서 원하는 상품 여러 개를 체크해요.
  2. 아래에 나타나는 프리셋 적용 버튼을 눌러요.
  3. 적용할 프리셋을 목록에서 선택해요.
  4. 선택한 프리셋에 이미지 처리 설정이 있으면 이미지도 함께 처리할지 물어봐요 — 원하는 대로 선택하면 돼요.
  5. 적용이 끝나면 몇 개가 성공했는지 결과를 보여줘요.

프리셋 관리

프리셋이란 자주 쓰는 설정을 미리 저장해뒀다가, 나중에 클릭 한 번으로 불러와 쓰는 기능이에요. 예를 들어 "쿠팡에 올릴 때 상품명 앞에 항상 붙이는 문구", "내가 자주 쓰는 가격 계산 공식" 같은 걸 미리 만들어두면, 새 상품을 편집할 때마다 같은 설정을 반복해서 입력하지 않아도 돼요.

새 프리셋 버튼을 눌러 이름을 정하면(예: "쿠팡 풀세트"), 아래 10개 항목 중 필요한 것만 채우면 됩니다. 전부 채울 필요는 없고, 채운 항목에는 "입력됨" 표시가 붙어요. 프리셋은 이름만 겹치지 않으면 몇 개든 만들 수 있어요.

1. 상품명

상품명을 새로 만들 때 쓸 AI 모델과 프롬프트(지시문)를 설정해요.

2. 옵션명

색상·사이즈 같은 옵션명을 번역할 AI 모델과 프롬프트를 설정해요. 번역 안함 체크박스로 번역을 아예 끌 수도 있고, 옵션이미지가 있는 속성에 순서대로 번호(1, 2, 3…)를 매기는 "옵션이미지 넘버링" 설정도 여기 있어요.

3. 상품 태그

항상 붙이고 싶은 고정 키워드를 등록해요. 개수 제한은 없어요.

4. 업로드 마켓

이 프리셋을 적용하면 어느 마켓 계정으로 업로드할지 미리 정해둘 수 있어요(체크박스로 여러 계정 선택 가능). 토스나 쿠팡 계정을 골랐다면, 그 마켓의 카테고리 구조에 맞게 옵션을 자동으로 변환할지("변환 안 함"/"토스쇼핑"/"쿠팡" 중 선택)도 함께 정할 수 있어요.

5. 계산식

가격 계산 공식(변수, 마진, 반올림 방식 등)을 미리 저장해둬요. 위의 "옵션 및 가격" 섹션에서 쓰는 계산기와 동일한 방식이에요.

참고 계산식이 잘못됐으면(예: 괄호가 안 맞음) "에러" 배지와 함께 화면 위쪽에 어떤 부분이 잘못됐는지 빨간 글씨로 알려주고, 고치기 전까지는 저장이 안 돼요. 만약 잘못된 계산식이 담긴 프리셋으로 상품이 수집되면, 상품 자체는 정상적으로 만들어지고 가격만 원가 그대로 들어오니 편집 화면에서 가격을 다시 확인해주세요.

6. 이미지

"상품 이미지(썸네일)"와 "옵션 이미지" 두 카드로 나뉘어요. 각각 "이미지 자동 수정"을 켜면, 배경제거·이미지번역·AI 이미지 변형(모델·프롬프트 선택 가능) 중 하나를 자동으로 적용하도록 설정할 수 있어요. 상품 이미지는 처리 결과를 "덮어쓰기"할지 "추가(원본 유지)"할지도 고를 수 있어요.

참고 이 자동 처리는 상품을 새로 수집하는 순간에만 적용돼요. 이미 수집된 상품에 이 프리셋을 나중에 적용해도 이미지는 그대로 유지됩니다.

7. 상세 설명

상세 설명 상단·하단에 자동으로 넣을 이미지 등을 설정해요.

8. 부가 정보

브랜드, 제조사, 출고일 등 "부가 정보" 섹션의 값들을 미리 정해둬요.

9. 금지어/대체어

이 프리셋을 적용한 상품에 금지어·대체어 규칙을 적용할지 여부만 켜고 끄는 곳이에요(체크박스 2개: 금지어 적용 / 대체어 적용). 실제 금지어·대체어 단어 목록 자체는 여기가 아니라 "금지어·대체어 관리" 메뉴에서 등록합니다.

10. 상품 속성

"수집 시 스마트스토어 속성 자동 매칭" 체크박스 하나예요. 켜두면 상품을 수집할 때 예측된 카테고리에 맞는 스마트스토어 상품 속성을 AI가 자동으로 골라줘요(정확하지 않을 수 있으니 편집 화면에서 검토하는 걸 추천해요). 이것도 수집 시에만 동작해요.

이렇게 만든 프리셋은 상품 편집 화면의 "프리셋" 섹션에서 클릭 한 번으로 적용하거나, 여러 상품에 한 번에 적용(위의 "일괄 프리셋 적용" 참고)할 수 있어요.

마켓 계정 설정

쿠팡, 스마트스토어 같은 마켓에 상품을 자동으로 올리려면, 그 마켓의 판매자 계정 정보를 ALL8에 미리 등록해둬야 해요. 이 화면에서 마켓별 계정을 추가하고 관리합니다.

  1. 등록하려는 마켓 탭(쿠팡/스마트스토어/ESM/11번가/롯데온/11번가글로벌)을 선택해요.
  2. 마켓 추가를 누르고 계정을 구분할 닉네임을 정해요.
  3. 그 마켓에서 발급받은 인증 정보(API 키 등)를 입력해요.
  4. 입력 후 연결 테스트 버튼을 누르면 실제로 그 마켓과 연결이 되는지 바로 확인할 수 있어요.
  5. 쿠팡은 출고지·반품지를 직접 입력하지 않고, 출고지 조회 / 반품·교환지 조회 버튼으로 이미 등록된 목록에서 골라 선택하면 돼요.
  6. 수수료율, 기본 배송비, 반품 배송비 같은 기본값도 함께 설정해요. 여기서 정한 값은 상품마다 배송 설정을 따로 입력하지 않아도 자동으로 적용돼요.
  7. 스마트스토어 계정은 원산지 조회 버튼으로 검색해서 계정 기본 원산지를 지정해둘 수 있어요. 상품마다 원산지를 따로 입력하지 않으면 이 기본값이 쓰여요.
  8. 저장을 누르면 완료예요.

ESM(G마켓/옥션) 계정 — 노옵션 카테고리 처리 방식

ESM 계정 설정에는 "옵션을 등록할 수 없는(노옵션)" 카테고리 상품을 어떻게 처리할지 정하는 항목이 있어요. 셋 중 하나를 고르면 돼요:

  • 개별 상품으로 등록 — 옵션 하나하나를 별개의 독립 상품으로 각각 등록해요.
  • 그룹으로 묶어 등록 — 옵션들을 하나의 상품 그룹으로 묶어서 등록해요.
  • 이 카테고리는 피하기 — 노옵션 카테고리 자체를 자동으로 피해서 다른 카테고리를 쓰도록 해요.
알아두면 좋아요 쿠팡·롯데온은 연결 테스트가 통과했다고 해서 인증 정보가 100% 맞다는 뜻은 아니에요. 확실하게 확인하고 싶다면 실제 상품을 하나 등록해보는 걸 추천해요.

금지어 · 대체어

상품명에 쓰면 안 되는 단어를 자동으로 지워주거나(금지어), 다른 단어로 자동 치환해주는(대체어) 기능이에요. 예를 들어 마켓 정책상 "정품"이라는 단어를 쓸 수 없다면, 이 단어를 금지어로 등록해두면 상품명을 저장할 때마다 자동으로 빠져요.

  1. 금지어 / 대체어 탭을 눌러 전환해요.
  2. 검색창으로 등록된 단어를 빠르게 찾을 수 있어요.
  3. 단어 추가로 한 개씩 추가하거나, 여러 개를 한 번에 입력해서 추가할 수도 있어요.
  4. 대체어는 나중에 연필 아이콘으로 수정할 수 있어요.
  5. 많은 양을 관리한다면 엑셀로 내보내고 가져오는 기능도 쓸 수 있어요.

적용 순서는 대체어로 먼저 바꾼 뒤, 남은 금지어를 지우는 순서로 자동 처리돼요. 대소문자 구분 없이 적용됩니다.

주문 관리

아직 준비 중이에요 마켓에서 들어온 주문을 확인하고 처리하는 기능은 현재 준비 중이에요. 곧 업데이트를 통해 추가될 예정입니다.

프롬프트 관리

AI가 상품명을 만들거나, 옵션명을 번역하거나, 이미지를 수정·생성할 때 따르는 "지시문(프롬프트)"을 미리 이름 붙여 저장해두는 곳이에요. 매번 같은 지시문을 새로 입력하지 않고, 저장해둔 걸 골라 쓸 수 있어요.

  1. 화면 위쪽 탭 4개 중 원하는 용도를 골라요: 상품명 프롬프트 / 옵션명 프롬프트 / AI 이미지 수정 / AI 이미지 생성. 프롬프트는 만들 때 고른 탭(용도)에 고정되고 나중에 바꿀 수 없어요.
  2. 프롬프트 추가 클릭 → 이름, (선택) 사용할 AI 모델, 프롬프트 내용을 입력해요.
  3. 저장하면 목록에 추가되고, 별표(⭐)를 눌러 그 용도의 "기본값"으로 지정할 수 있어요(용도마다 기본은 1개만).
  4. 연필 아이콘으로 수정, 휴지통 아이콘으로 삭제할 수 있어요.

목록 표에는 추가일과 수정일 칸도 있어서, 언제 만들고 언제 마지막으로 고쳤는지 바로 확인할 수 있고, 칸 제목을 클릭하면 최신순/과거순으로 정렬돼요.

프롬프트를 파일로 주고받기 — 공유파일 다운로드/업로드

다른 사람(팀원 등)과 프롬프트를 파일로 주고받을 수 있는 기능이에요. 공유파일 다운로드로 지금 목록에 있는 프롬프트를 전용 파일(.a8p)로 내려받고, 상대방은 공유파일 업로드로 그 파일을 불러와 자기 프롬프트 목록에 추가할 수 있어요. 이 파일은 이름과 프롬프트 내용만 담고(어떤 프롬프트가 기본값인지, 생성·수정 날짜 같은 개인 기록은 담기지 않아요) 내용을 그대로 텍스트로 열어볼 수 없게 인코딩되어 있어요 — 개인정보 백업용이 아니라 팀 간 공유 전용 기능이에요.

저장한 프롬프트는 어디서 쓰나요

상품 편집 화면의 상품명·옵션명 "↻ 재번역" 패널이나, 이미지 섹션의 "AI 수정"/"AI 생성" 창을 열면 내 프롬프트 ▾ 드롭다운이 있어요. 여기서 저장해둔 프롬프트를 바로 불러올 수 있고, 반대로 지금 입력한 프롬프트를 그 자리에서 바로 이름 붙여 저장할 수도 있어요.

참고 상품명·옵션명 프롬프트는 처음 이 화면에 들어가면 기본 프롬프트가 자동으로 하나씩 만들어져 있어요 — 아무것도 안 만들어도 바로 쓸 수 있습니다. 다만 AI 이미지 수정·생성은 기본 프롬프트가 없어서 직접 만들어야 해요. 그리고 프리셋의 상품명/옵션명 설정에서 프롬프트를 아직 안 골랐다면, 여기서 지정한 기본(⭐) 프롬프트가 자동으로 연결돼요 — 나중에 여기서 원본을 수정하면 프리셋에도 그대로 반영됩니다(단, 프리셋에서 직접 프롬프트 내용을 고치면 그 순간부터는 따로 관리돼요).

결제내역

크레딧 충전, 구독료, 환불 같은 결제 기록을 모아서 볼 수 있는 화면이에요. 사이드바가 아니라 화면 오른쪽 위 프로필 메뉴를 눌러야 들어갈 수 있고, 워크스페이스를 처음 만든 소유자 계정에게만 보여요(직원 계정으로 로그인하면 이 메뉴 자체가 나타나지 않아요).

목록에는 결제 종류(크레딧 충전 / 구독료 / 운영자 청구 / 정산)와 금액, 날짜가 표로 나열돼요. 별도의 검색이나 기간 필터는 없고, 전체 내역이 최신순으로 쭉 보여요.

환불 신청 — 2단계로 진행돼요

결제 건 옆의 환불 버튼을 누르면 바로 돈이 돌아오는 게 아니라, 아래 순서로 진행돼요:

  1. ① 정산 청구서 발행 요청 — 먼저 환불을 요청하면, 운영팀이 확인할 수 있도록 정산 청구서가 발행돼요.
  2. ② 원결제 전액취소 — 확인이 끝나면 처음 결제했던 수단으로 전액이 취소돼요.

환불 처리에는 시간이 조금 걸릴 수 있으니, 신청 후 바로 반영되지 않아도 정상이에요.
